# Patrick Bain Web Services Website

Multi-page static site built with HTML, CSS, and JavaScript.

## Pages
- Home: /index.html
- Services: /pages/services.html
- Maintenance: /pages/maintenance.html
- Contact: /pages/contact.html
- Privacy: /pages/privacy.html
- Terms: /pages/terms.html

## Contact Form (Formspree)
1. Create a Formspree form
2. Copy your form endpoint ID
3. Replace `YOUR_FORMSPREE_ID` in `pages/contact.html`

## Analytics (optional)
In `index.html`, replace `G-XXXXXXXXXX` with your Google Analytics Measurement ID.

## Deploy (GitHub Pages)
1. Push the repo to GitHub
2. In GitHub: Settings → Pages → Deploy from branch
3. Select main branch and root
4. Add your custom domain (patrickbainweb.com)
5. Ensure HTTPS is enabled

## Notes
- Keep copy business-focused, not technical.
- Avoid placeholders before outreach.
