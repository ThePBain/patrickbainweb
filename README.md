# patrickbainweb
Business website for **Patrick Bain Web Services**. Built with modern frontend practices to demonstrate clean, performant, and professional web design for small businesses.

## Overview
This repository contains the source code for `patrickbainweb.com`, a code-first, single-page business site focused on:
- Clarity and trust for non-technical visitors
- Clean, premium visual design
- Strong performance and responsive behavior
- Reusable structure that can scale into future client projects

## Core Features
- **Responsive layout** designed mobile-first
- **Performance-focused** assets and structure
- **Accessible, semantic HTML** for reliability and SEO
- **Polished interaction details** (navigation, scroll behavior, form UX)
- **Reusable code organization** for quick iteration and future builds

## Tech Stack
- **HTML5**
- **CSS3**
- **Vanilla JavaScript**

No frameworks by design. The goal is speed, control, and demonstrating strong fundamentals.
